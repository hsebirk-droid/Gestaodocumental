## 📋 Sobre o Projeto

O **e-coordina** é uma plataforma web para gestão documental e coordenação de atividades empresariais, desenvolvida em conformidade com a Lei nº 102/2009 e o Código do Trabalho português.

Esta demonstração interativa apresenta as principais funcionalidades da plataforma, permitindo explorar:

- Gestão de empresas, trabalhos e centros de trabalho
- Pedidos de documentação e validação
- Controlo de acessos em tempo real (entradas/saídas)
- Coordenações e cadeia de subcontratação
- Módulo PRP (Ações corretivas e comunicações de risco)
- Ativação de novos fornecedores
